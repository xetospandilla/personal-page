# Personal Page - Junior Developer

Mi portafolio personal como Programador Junior. Una página web donde muestro mis proyectos, habilidades y experiencia en desarrollo web.

## 🎯 Objetivo

Crear un portafolio profesional que muestre mis habilidades técnicas, proyectos realizados y facilite el contacto con potenciales colaboradores o empleadores.

## 📋 Secciones

- **Hero/Inicio**: Presentación personal con animaciones
- **Sobre Mí**: Experiencia, proyectos completados y skills
- **Proyectos**: Timeline horizontal interactivo con mis trabajos
- **Contacto**: Links a redes sociales y formas de comunicación

## 🛠️ Tecnologías

- **HTML5**: Estructura semántica
- **CSS3**: Estilos y animaciones
- **Responsive Design**: Optimizado para móvil, tablet y desktop
